# Application de Gestion de Produits (Django 6.0)
Ce projet est une application web simple permettant d'afficher et d'ajouter des produits avec images, développée avec Django 6.0 et une base de données MySQL.

## Prérequis

- Python 3.10 ou supérieur
- MySQL (Laragon, XAMPP ou MySQL Server)
- Git

## Installation
Suivez ces étapes pour installer le projet sur une nouvelle machine :
- git clone https://github.com/jihaneelaouni/Django-app-product.git
- pip install mysqlclient
- python manage.py makemigrations
- python manage.py migrate
- python manage.py runserver
